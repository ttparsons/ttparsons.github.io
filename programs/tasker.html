What is Tasker?

Tasker is a very sophisticated Android application that allows you to run complicated scripts on your phone. You can program it to run “tasks” when certain “profiles” are active.

For example, you can create a profile that turns your WiFi off when you leave your house. Or, you can create a profile that says the current traffic conditions out loud when you start your car (See my below tutorials!). It is very easy to learn once you figure out how it works. It is a fun way to make your life a little easier.
